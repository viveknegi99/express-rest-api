# Devcamper API

> Backend API for Devcamper application which is bootcamp directory website

## Usage

Rename config.env.env to config.env and update the value to your own

## Install dependecies

```
npm install
```
## Run app
```
# Run in dev mode
npm run dev

# Run in production mode
npm start
```
